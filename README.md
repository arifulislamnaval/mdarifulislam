# 🛥️ Fish Research Vessel (FRV)

## Overview
This Fish Research Vessel (FRV) was built for the Bangladesh Fisheries Research Institute (BFRI) to support fisheries research, fish stock assessment, and marine data collection activities.

---

## Principal Particulars

| Particular | Specification |
|------------|--------------|
| Length Overall (LOA) | 25.0 m |
| Breadth (Moulded) | 6.0 m |
| Depth (Moulded) | 2.8 m |
| Draft | 1.4 m |
| Maximum Speed | 10 Knots |
| Special Features | Fish Finder System & Fully Equipped Fish Research Laboratory |

## Key Features

- Fish Finder System for locating fish schools and monitoring fish movements.
- Dedicated Fish Research Laboratory onboard.
- Marine biological sample collection and analysis facilities.
- Suitable for coastal and inland fisheries research operations.
- Reliable twin-engine propulsion arrangement.
- Designed to comply with Department of Shipping (DOS) requirements.
